# Css Components

If you don't use VueJS for your application, you can use the CSS only version of our components. It includes all the
styles used in the VueJs component library.

## Usage

1. Import the full css bundle or the css for a specific component

```css
/* Full bundle */
@import '@prestashopcorp/puik/dist/index.css';
/* Specific component */
@import '@prestashopcorp/puik/theme/base.css';
@import '@prestashopcorp/puik/theme/puik-button.css';
```

2. Add the classes in your HTML

```html
<button class="puik-button puik-button--primary">Example button</button>
```

<h1 id="design-tokens">Css Design Tokens</h1>

If you want to use the design tokens used in this library you can import the full design tokens bundle

```css
@import '@prestashopcorp/puik/theme/design-tokens';
```

Or if you need specific design tokens like the colors for example you can import the specific file (specifc files are prefixed by `puik-`)

```css
@import '@prestashopcorp/puik/theme/design-tokens/puik-colors';
```
